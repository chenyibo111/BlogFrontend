### #13 CSP 头缺失 ⏳

**问题**: 未配置 Content-Security-Policy

**建议**:
```typescript
// Backend middleware
app.use((req, res, next) => {
  res.setHeader(
    'Content-Security-Policy',
    "default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline' https://fonts.googleapis.com"
  );
  next();
});
```

### #17 Role 字段类型 ⏳
